⚙ Integrations

API keys are stored locally in your browser — never sent to any server.

Google

Client ID + Secret are used for Google Drive, Contacts, and Calendar. The secret enables long-lived Calendar sessions (no more reconnecting). Get both at console.cloud.google.com → APIs & Services → Credentials.

Anthropic

Used for AI-assisted order scanning and estimates.

Notion

The Notion API token is stored as a server-side environment variable in Cloudflare Pages — not in your browser. It is the source of truth for the Order Pipeline, Notes, Customers, and Supplier Receipts.

↗ Open Cloudflare Pages Dashboard

Go to your Pages project → Settings → Environment Variables → set NOTION_TOKEN to your integration secret. Get the token at notion.so/profile/integrations.

Make sure your integration is connected to both the Custom Orders and STS Notes databases in Notion (open each database → … → Connections → add your integration).

ShipStation

Used to look up tracking numbers by order # (🔎 button next to Tracking Number fields). The API key/secret are stored as server-side environment variables in Cloudflare Pages — not in your browser.

↗ Open Cloudflare Pages Dashboard

Go to your Pages project → Settings → Environment Variables → set SHIPSTATION_API_KEY and SHIPSTATION_API_SECRET. Get both at ShipStation → Account Settings → API Settings.

Square

Get both at developer.squareup.com → Your App → Credentials (Production). Location ID is auto-detected from your token when you save.
