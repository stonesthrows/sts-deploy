⚙ Integrations

API keys are stored locally in your browser — never sent to any server.

Google

Used for Google Drive bag scans and Contacts sync. Create one at console.cloud.google.com.

Anthropic

Used for AI-assisted order scanning and estimates.

ClickUp

Get your token at app.clickup.com → Settings → Apps → API Token.

Notion

The Notion API token is stored as a server-side environment variable in Cloudflare Pages — not in your browser. It is already used for the Custom Orders integration. If notes are not loading, verify the token is set there.

↗ Open Cloudflare Pages Dashboard

Go to your Pages project → Settings → Environment Variables → set NOTION_TOKEN to your integration secret. Get the token at notion.so/profile/integrations.

Also make sure the STS Notes database in Notion has your integration connected: open the database → … → Connections → add your integration.

Square

Used for live inventory sync. Get a token at developer.squareup.com → Your App → Credentials. Use a Production token for live data.
